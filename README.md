# LAN 的个人博客与工具站

这里是我写技术笔记、折腾各种小工具的地方。所有内容完全免费，欢迎随意使用。

## 📁 项目结构

```
.
├── index.html          # 首页
├── style.css           # 全局样式
├── find.html           # 查找页面
├── jump.html           # 跳转页面
├── blog/               # 博客目录
│   ├── index.html      # 博客列表页
│   └── posts/          # 博客文章
├── tools/              # 工具目录
│   ├── index.html      # 工具导航页
│   ├── calculator.html # 计算器
│   ├── minesweeper.html# 扫雷游戏
│   ├── snake.html      # 贪吃蛇游戏
│   ├── todo.html       # 待办事项
│   ├── tomato.html     # 番茄钟
│   └── weather.html    # 天气查询
└── data/               # 数据目录
```
